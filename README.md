# 📸 Photo Editor

A modern Photo Editor built using OpenCV and Streamlit with an interactive UI.

---

## 🚀 Features

* 📤 Upload image
* 📐 Resize (width & height with aspect ratio)
* 🎚️ Brightness & contrast adjustment
* 🎨 Filters:

  * Blur
  * Sharpen (Unsharp Masking)
  * Warm filter
  * Grayscale
* 🧠 Portrait background blur
* 🖼️ Before vs After comparison
* 📥 Download edited image

---

## 🛠️ Tech Stack

* Python
* OpenCV
* Streamlit
* NumPy
* PIL

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 💡 Key Concepts Used

* Image Processing (OpenCV)
* Interpolation Techniques
* Unsharp Masking (Sharpening)
* UI Design with Streamlit
* Real-time Image Manipulation
